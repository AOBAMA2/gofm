Instrucciones=

Si deseas correr la aplicación en el puerto 80 solo debes:
iniciar el programa, 
teclea 1
teclea enter


Si deseas correr la aplicación en otro puerto solo debes:
iniciar el programa, 
teclea 2
teclea el puerto con dos digitos y mayor que 80
teclea enter
